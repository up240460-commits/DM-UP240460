1. ¿Qué es React? React es una biblioteca de JavaScript de código abierto desarrollada por Facebook, Se utiliza para construir interfaces de usuario (IU) interactivas y eficientes para aplicaciones web. Su característica principal es que permite dividir la interfaz en piezas independientes y reutilizables. Funciona mediante un "DOM Virtual" que optimiza la actualización de pagina, haciendo que las aplicaciones sean muy rápidas.

2. ¿Qué es React Native? Es un framework de código abierto, también creado por Meta, que permite crear aplicaciones móviles nativas para iOS y Android utilizando JavaScript y React. A diferencia de React (web), React Native no usa HTML ni CSS estándar; es su lugar utiliza componentes que se traducen (renderizan) en la interfaz nativa del teléfono, logrando que la aplicación se sienta y comporte como una app nativa real, no como una pagina web incrustada.
