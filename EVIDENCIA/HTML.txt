1. ¿Que es html y para que sirve?
HTML (HyperTex Makuo Language) es un lenguaje de marcado que se usa para crear y estructurar paginas web.
Sirve para definir el contenido de una página, como textos, imágenes, enlaces, listas, tablas, etc.

2. ¿Cuál es la diferencia entre html, ccs y JavaScript? 
HTML: estructura el contenido (que hay en la pagina).
CSS: de estilo y deseño (colores, tamaños, posiciones).
JavaScript: añade interacción y comportamiento (botones, animaciones, validaciones).

3. ¿Por que html no es un lenguaje de programación?
Porque no puede realizar operaciones lógicas, como condiciones (if), ciclos (while) o cálculos.
HTML solo describe contenido, no ejecuta instrucciones ni toma decisiones.

4. ¿Cuál es la estructura básica de un archivo html?

<!DOCTYPE html>
<html>
  <head>
    <title>Mi página</title>
  </head>
  <body>
    <h1>Hola mundo</h1>
  </body>
</html>

- <!DOCTYPE html>
Indica que el documento está escrito en HTML5.
Ayuda al navegador a interpretar correctamente la página.
- <html>
Es la etiqueta principal del documento.
Contiene todo el contenido de la página web.
- <head>
Contiene información interna de la página.
No se muestra directamente al usuario.
Aquí van metadatos, enlaces a CSS, el título, etc.
- <title>
Define el título de la página.
Es el texto que aparece en la pestaña del navegador.
- </head>
Cierra la sección de información del documento.
- <body>
Contiene todo lo que el usuario ve en la página.
Aquí van textos, imágenes, botones, enlaces, etc.
- <h1>
Es un encabezado principal.
Se usa para títulos importantes.
Existen desde <h1> hasta <h6> (de mayor a menor importancia).

5. ¿Qué es una etiqueta en html, especifica e identifica cual es la etiqueta de cierre, apertura y contenido?
Una etiqueta sirve para definir un elemento en HTML. 
<p>Hola</p>
Etiqueta de apertura: <p>
Contenido: Hola
Etiqueta de cierre: </p>

6. ¿Qué es un atributo en html y para que sirve?
Un atributo proporciona información adicional sobre una etiqueta.
Sirve para modificar su comportamiento, apariencia o identificación 

<img src="foto.jpg">


7. ¿Qué es el atributo class?
El atributo class se usa para agrupar elementos y aplicarles estilos o comportamientos iguales con CSS o JavaScript.

<p class="texto">Hola</p>


8. ¿Que es el atributo ID?
El atributo id identifica un elemento de forma unica dentro de la pagina.
<div id="menu"></div>


9. ¿Cómo se escribe un atributo de escritura, pon un ejemplo
Un atributo se escribe dentro de la etiqueta de apertura, con esta estructura:
<a href="https://www.google.com">Ir a Google</a>
